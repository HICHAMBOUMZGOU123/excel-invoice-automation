# Excel Invoice Automation

## Description
Projet Python permettant d’automatiser le traitement de fichiers Excel.
Le script vérifie la conformité des fichiers, nettoie les données et génère automatiquement des fichiers de facturation par client.

## Technologies
- Python
- openpyxl

## Structure
- main.py : script principal
- data/modele1.xlsx : fichier Excel exemple (format attendu)

## Exécution
```bash
python main.py
